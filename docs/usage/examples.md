# Examples

## Build examples

This example runs a simple NodeJS web site demonstrating scenarios of usage DP WAM from a browser.

## Run examples

To run from terminal:

``` bash
cd ./examples/browser
npm i
npm start
```

Browse to http://localhost:3000/.
